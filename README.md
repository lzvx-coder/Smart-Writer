# 智能写作与批改助手

## 一、软件简介

**智能写作与批改助手** 是一款基于大语言模型的本地桌面应用（Windows EXE），集 **多语言润色、智能批改、文本优化、版本对比** 于一体，适用于学生写作、课程作业、论文初稿、技术文档与日常文本处理等场景。

本程序由前端 HTML + JavaScript 构建，并已通过打包工具封装为 **Windows 可执行文件（.exe）**，无需浏览器即可直接运行。

---

## 二、主要功能

### 🌐 多语言润色

* 支持中文 / 英文等多语言输入
* 可一键转换为以下写作风格：

  * 学术论文
  * 商务邮件
  * 技术文档
  * 社交媒体文案
* 在 **不改变原意和原语言** 的前提下提升表达质量

### 📝 智能批改

* 支持多类型文本：

  * 学术论文
  * 商务文档
  * 技术文档
  * 创意写作
  * 通用文本
* 提供：

  * 总分评分
  * 多维度评分（语法、逻辑、清晰度、风格、创新性）
  * 问题定位与修改建议
  * 总体评价总结

### ✨ 风格优化

* 可按需勾选优化方向：

  * 提升清晰度
  * 简洁表达
  * 增强专业性
  * 提升吸引力
* 自动生成优化后的完整文本

### 🔄 版本对比

* 对比两个文本版本
* 分析：

  * 主要差异点
  * 各自优缺点
  * 推荐使用场景
  * 改进建议

### 📜 本地历史记录

* 自动保存最近使用记录（本地浏览器存储）
* 支持一键回溯历史内容

## 四、使用方法

### 1️⃣ 启动程序

双击运行：

```
智能写作与批改助手.exe
```

首次启动可能会被 Windows 安全提示拦截，请选择：

> 更多信息 → 仍要运行

---

### 2️⃣ 设置 API Key（重要）

本软件需要 **大模型 API Key** 才能正常工作。

如你使用的是 **源码版重新打包**，请在源码中找到以下位置并填写你的 Key：

```js
const API_KEY = '你的API Key';
```

如果你使用的是 **已配置好的 EXE 成品版**，可直接使用，无需额外配置。

⚠️ 请妥善保管 API Key，避免泄露。

---

### 3️⃣ 功能使用流程（示例）

以「多语言润色」为例：

1. 选择写作风格（如：学术论文）
2. 在左侧输入文本或上传 Word / TXT 文件
3. 点击【生成内容】
4. 在右侧查看结果
5. 可复制或下载生成文本

其他功能模块操作方式类似。

## 六、常见问题

### Q1：点击生成后无反应？

* 请检查网络是否正常
* 请确认 API Key 是否有效
* 若频繁请求，可能触发接口限流，稍后再试

### Q2：评分或批改结果解析失败？

* 偶发模型输出异常，重新点击「开始批改」即可

### Q3：历史记录会同步到云端吗？

* 不会。历史记录仅保存在 **本地设备**，清除浏览器数据或重装程序后会丢失。
